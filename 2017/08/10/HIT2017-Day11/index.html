BZOJ3224普通平衡树

题目大意:

支持: 1.插入x数 2.删除x数(若有多个相同的数，因只删除一个) 3.查询x数的排名(若有多个相同的数，因输出最小的排名) 4.查询排名为x的数 5.求x的前驱(前驱定义为小于x，且最大的数) 6.求x的后继(后继定义为大于x，且最小的数)

CODE

BZOJ4864神秘物质

题目大意:

对于一个序列支持: 插入,删除 询问一个区间内所有长度大于$1$的区间的最大,最小极差

题解:

最大极差维护区间最大值最小值即可 一个性质:最小极差一定为相邻两个数差的绝对值 证明:如果左右扩展出更大或更小不会更优,如果不大也不小会更新但也是相邻两个

CODE

BZOJ1500维修数列

题目大意:

要求支持插入,删除,翻转,区间整体赋值,求一段的和,求区间最最大子段和

题解:

经典题,维护子树的和,子树最大字段和,左端点开始最大子段和,右端点结尾最大子段和,翻转标记,区间整体赋值标记

CODE

BZOJ2809

题目大意:

给定一个树结构,每个点有两个值$a_i,b_i$,要找到一个节点$pos$作为父节点,在他及其子树中找到尽可能多的节点$(cnt$个点$)$,但 $\sum{b_i}<m$ ,询问 $max(a_{pos}*cnt)$

题解:

子树尽可能选多每次尽可能取小的即可 这样我们枚举父节点,用可并堆处理即可 左偏树内部儿子指针指来指去,外部用一个指针指到根顶

CODE

BZOJ3196二逼平衡树

题目大意:

1.查询k在区间内的排名 2.查询区间内排名为k的值 3.修改某一位值上的数值 4.查询k在区间内的前驱(前驱定义为小于x，且最大的数) 5.查询k在区间内的后继(后继定义为大于x，且最小的数)

题解:

分块大法好 不涉及插入,分块大法好 3.修改只要每次rebuild即可 1.查询区间内比k小的数 2.二分答案,查询排名判断 4.查询区间内比x小最大的数 5.查询区间内比x大最小的数 $O(mlogX_i\sqrt{n}log{n})$

CODE
